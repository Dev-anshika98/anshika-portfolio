# anshika-portfolio
Welcome to my personal website, showcasing my projects and skills in web development..

## :information_source: Requirements

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)

# Process to Setup 

**To install this project**

```
npm install
```

**To run the server**

```
npm start
```

**To build the app**

```
npm run build
```
